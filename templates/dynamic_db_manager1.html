🔍 Auto-Discovery Information

This system automatically detects databases based on filename patterns:

📚 QBank Databases:
*year*.db
Examples: 1st_year.db, 2nd_year.db, 3rd_year.db
📝 MCQ Databases:
*mcq*.db
Examples: anatomy_mcq.db, cardiology_mcq.db
👤 Admin Databases:
admin*.db
Examples: admin.db, admin_settings.db
💡 Tips:
  • New databases are automatically discovered when you restart the application
  • Upload existing .db files using the "📤 Upload Database" button
  • Use "🔍 Debug" to troubleshoot database access issues
  • Regular backups are recommended before making major changes
